# Farah Cisse Portfolio

Static portfolio site, showcasing work for data science, ML, and AI opportunities.

## Run locally

Because this is a plain static site, you can open `index.html` directly in a browser or serve it with any simple static server.

If you want a local server from this folder:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

This project deploys easily on:

- GitHub Pages (Currently deployed here)
- Netlify
- Vercel

## Customization notes

- Update the portrait at `assets/farah-portrait.jpeg`
- Update the resume at `assets/Farah_Cisse_Resume.pdf`
- Edit chatbot answers in `script.js`
- Edit copy and sections in `index.html`
- Tweak colors and layout in `styles.css`
